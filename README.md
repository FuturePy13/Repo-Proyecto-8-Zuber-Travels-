“Repository containing a Data Analysis project with Pandas and included visualizations with Matplotlib, hypothesis resolution and variation values with Stats Library. Also using my own SQL querys to work with the data in the Jupyter Notebook”

Este proyecto se desarrolla como parte del trabajo para Zuber, una nueva empresa de viajes compartidos que busca posicionarse en la ciudad de Chicago.
El objetivo principal es identificar patrones clave en los datos disponibles, comprender el comportamiento de los pasajeros y analizar cómo los factores externos —especialmente el clima— influyen en la frecuencia de los viajes.

El análisis también incluye una evaluación de datos de competidores, con el fin de ofrecer a Zuber insights estratégicos para mejorar su lanzamiento al mercado.

Objetivos del Proyecto

* Analizar el comportamiento y preferencias de los pasajeros.

* Estudiar patrones temporales (por hora, día de la semana, temporada).

* Evaluar el impacto del clima en la demanda de viajes.

* Comparar datos de competidores para entender el mercado local.

* Probar una hipótesis clave:
  “Las condiciones climáticas afectan la frecuencia de los viajes.”
